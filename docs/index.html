DotMenu

A macOS menu bar utility for quick screen captures with annotation tools.

Download .dmg GitHub

Features

Three capture modes

Selection, full screen, or foremost window — triggered from the menu bar or global hotkeys.

Drawing annotations

Rectangle, circle, line, and arrow tools with adjustable color, fill, and line width.

Copy & Save

Copy annotated captures to the pasteboard (Cmd+C, Enter) or save as PNG with automatic naming.

Color palette extraction

K-Means extracts 5 dominant colors from the capture. Click a swatch to set drawing color and copy its hex value.

Configurable hotkeys

Global shortcuts for all three capture modes — defaults suit both QWERTY and AZERTY layouts. Rebiddable in Preferences.

Capture history

Last 10 captures persist across app restarts. Access them directly from the menu bar, with annotations preserved.
